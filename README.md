# Ink & Leaf - A Modern Bookstore Experience

A premium React-based bookstore website with elegant design and modern user experience.

## Features

- **Premium Design**: Elegant and sophisticated UI with warm color palette
- **User Authentication**: Login and registration system
- **Book Catalog**: Browse books by categories with search functionality
- **Shopping Cart**: Add books to cart and manage purchases
- **Responsive Design**: Works perfectly on desktop and mobile devices
- **Modern UX**: Smooth animations and intuitive navigation

## Tech Stack

- React 18
- React Router DOM
- CSS3 with modern features
- Local Storage for data persistence
- Google Fonts (Playfair Display & Inter)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
   ```bash
   cd "book shop"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open your browser and visit `http://localhost:3000`

## Usage

1. **Browse Books**: Visit the homepage and explore featured books
2. **Create Account**: Click "Get Started" or "Login" to create an account
3. **Shop Books**: Navigate to the Books page to browse the full catalog
4. **Add to Cart**: Click "Add to Cart" or "Buy Now" on any book
5. **Manage Cart**: View and manage your cart items
6. **Checkout**: Proceed to checkout from the cart page
